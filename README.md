# 🧮 Calculadora em Python

Uma calculadora simples feita em Python, criada como primeiro projeto de programação.

O objetivo deste projeto é praticar os fundamentos da linguagem Python, como entrada de dados, funções, validação de respostas e organização de código.

---

## 📌 Sobre o projeto

Este projeto começou como uma calculadora básica no terminal.

No momento, o programa permite que o usuário escolha uma operação matemática por meio de um menu numérico. A ideia é evoluir o projeto aos poucos, adicionando novas funcionalidades conforme o aprendizado avança.

---

## 🚀 Funcionalidades atuais

- Exibe um menu de operações matemáticas.
- Recebe a opção digitada pelo usuário.
- Valida se a opção escolhida é válida.
- Converte a resposta para número inteiro quando válida.
- Mostra uma mensagem caso a opção seja desconhecida.
